# Reliance-Sales-PowerBI-Dashboard
Retail Sales Performance Dashboard using Power BI
This project analyzes retail sales performance using Power BI.

## Key Insights
- Region-wise Revenue Analysis
- Category-wise Sales Contribution
- Profit Margin Analysis
- Monthly Revenue Trends

## Dashboard Features
- Interactive KPI Cards
- Treemap, Map, Scatter and Line Charts
- Advanced DAX (YTD, QTD, MoM Growth)

## Tools Used
- Power BI
- DAX
- Data Visualization
